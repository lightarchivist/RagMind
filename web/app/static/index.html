RAG-MIND
RAG-MIND is a fully local, privacy-first document Q&A system. Users upload PDFs, EPUBs, and other documents, index them using a local embedding model, and ask natural language questions answered exclusively from document content — no cloud services, no API keys, no data leaving the machine.
🦙 Llama 3 🔍 nomic-embed-text ⚡ FastAPI 🐍 Python 🐳 Docker 🔒 100% Local
